# Sistema de Gestão de Doações - ONG Mãos Solidárias

## Descrição

Projeto desenvolvido para a disciplina Projeto Integrador Extensionista I (PIE I).

O sistema tem como objetivo auxiliar a ONG Mãos Solidárias no cadastro de doadores, registro de doações e geração de relatórios.

## Sobre a ONG

A ONG Mãos Solidárias atua em Belo Horizonte/MG, arrecadando e distribuindo alimentos, roupas e produtos de higiene para famílias em situação de vulnerabilidade social.

## Estrutura do Projeto

- /backend
- /frontend
- /database
- /docs

## Instalação

1. Clone este repositório.
2. Configure o arquivo `.env` utilizando o `.env.example`.
3. Instale as dependências do projeto.
4. Execute a aplicação.
